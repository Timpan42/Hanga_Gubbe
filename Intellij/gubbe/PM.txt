Hänga gubbe program

inledning
Jag skrev ett program som skulle kopiera spelet hänga gubbe. Mitt arbetssätt va att dela upp dem stora problemen i mindre delar och göra dem steg för steg.  Genomförandet gjordes i code språket java. Genom att göra deta projekt har jag lärt mig att förstå bättre hur man delar up problem i mindra delar. 

bakrund 
Jag började att dela up hänga gubbet spelet i mindre delar. Så att jag hade mindre problem att lössa i dem olika delarna, istället för att tänka hella spelet som ett enda stort problem som skulle lösas på en gång. Det skulle göra codnings dellen änklare för man behövde bara lösa små problem som kondens senare setas ihop och hoppningsvis så funkar programet. Tesnings delen av programmet var jag som körde programmet och kollade om det funkade eller inte och letade efter problem. 

posetiv
Det som har gått bra under projektets gång har varit uppdelningen av programmet och att söka upp information på hur vissa problem kan lössas genom google eller lärare.
Jag tycktet det var enklet att dela upp hänga gubbe spelet i dess olika problem så man skunde enklare lössa dem senare. Det relativt bra att söka upp information om problem jag hade och senare lösa dem med hjälp av den informationen.

negativ
Det som var dåligt har varit få en start på projektet, skriva rätt kode.
Jag hade svårt att börja projektet med hur jag skulle lössa vissa av dem problemne som fanns. För att undvika deta i framtiden så skulle det vara bra att minnas vissa av dem vanligaste kode bitarna som används. Jag hade också svårt med att hita rätt kode som jag skulle använda för att lössa vissa problem. Som exempel att lössa problemet att få underscore som representera ett ords bokstäver. Kan fråga oftare om hjälp och prata med någon annan för att lära sig av den eller för att bolla ider. 

sammanfatning
Jag har lär mig att jag kuna träna lite mer på grunderna av kodning. Och att kuna ta risker med hur jag skriver koden bara för att se om det funkar eller inte. Programet är inte nära klart dock är den en start på hur ett fungerande hänga gubbe spel kan se ut.
